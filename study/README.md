# 형태소 분석
## 형태소와 어절
### 형태소
- 일정한 의미가 있는 가장 작은 말의 단위(즉, 더 분석하면 뜻이 없어지는 말의 단위)

### 어절
- 한국어에서 발음의 기본이 되는 단위(대개 띄어쓰기의 단위와 일치)

## Khaiii의 등장
- cnn모델 기반
- rnn, lstm 은 순차적 계산으로 많은 계산량으로 인해 느림
- 규칙과 확률이 아니라 **학습** 으로 품사 결정(데이터 기반-세종코퍼스)
